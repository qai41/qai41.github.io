第一部分：油猴脚本安装与使用

1安装油猴扩展

1

安装 Tampermonkey

在浏览器扩展商店搜索 Tampermonkey（油猴）并安装。支持 Chrome、Edge、Firefox 等主流浏览器。

官网地址：https://www.tampermonkey.net/

官网下载：https://www.tampermonkey.net/crx/tampermonkey_stable.crx

2

确认扩展已启用

安装后浏览器工具栏会出现油猴图标，点击可查看已安装的脚本列表。

2安装脚本

1

下载脚本文件

点击下方按钮下载脚本文件 a-bao-auto-fill.js

⬇️ 下载脚本文件
2

导入到油猴

点击油猴图标 → 「管理面板」→ 「实用工具」→ 「导入」→ 选择下载的脚本文件，或直接将脚本文件拖拽到浏览器窗口。

3

保存脚本

脚本编辑页面打开后，点击「文件」→「保存」或按 Ctrl+S 保存。保存成功后脚本会自动运行。

3验证安装

1

查看悬浮按钮

打开目标网页后，页面右侧会出现一个蓝色的「粘贴 JSON 填充」悬浮按钮，说明脚本安装成功。

💡 提示
如果看不到悬浮按钮，请检查：1. 油猴扩展是否已启用 2. 当前网址是否匹配 3. 浏览器控制台是否有错误

第二部分：脚本功能使用方法

1配置字段映射规则

首次使用需要先配置 JSON 数据与表单字段的映射关系。

1

打开配置面板

在目标网页上，右键点击蓝色的悬浮按钮，选择「⚙️ 打开配置」，即可看到规则编辑面板。

规则配置面板界面
2

填写规则信息

  • 规则名称：给这条规则起个名字，方便识别
  • 自动匹配网址（可选）：填写网址关键词，脚本会自动匹配适用该规则的页面
3

配置字段映射

在「字段映射」区域，每一行代表一个映射关系：

列名 说明 示例
第一列 JSON 中的字段名（key） json_key_1
第二列 表单中的目标字段名称 身份证正反面照片
第三列 字段类型（图片/文本/下拉框/文本域） 图片
4

添加/删除映射行

点击「+ 添加一行」增加新的映射，点击每行右侧的「×」删除该行。点击「清空」可删除所有映射。

5

保存规则

配置完成后，点击右下角的 保存规则 按钮保存。也可以先点击 预览匹配 查看配置是否正确。

2使用自动填充功能

1

准备 JSON 数据

将要填充的数据整理成 JSON 格式，复制到剪贴板。例如：

{
  "json_key_1": "https://example.com/idcard.jpg",
  "json_key_2": "https://example.com/avatar.jpg",
  "json_key_3": "https://example.com/form.jpg"
}
2

打开目标弹窗

在网页上点击按钮打开需要填充的影像采集弹窗（通常是包含图片上传区域的对话框）。

3

执行填充

点击右侧的「粘贴 JSON 填充」按钮，脚本会自动：

  • 读取剪贴板中的 JSON 数据
  • 根据配置的映射关系找到对应表单字段
  • 显示填充成功的提示
💡 手动粘贴模式
如果浏览器限制剪贴板读取，脚本会自动弹出输入框，让你手动粘贴 JSON 数据。

3管理多个规则

1

打开规则管理

右键点击悬浮按钮，选择「📋 规则管理」，可以查看所有已保存的规则。

2

规则操作

  • 编辑：修改已有规则的配置
  • 删除：删除不需要的规则
  • 切换：点击规则名称切换当前使用的规则
3

导入/导出规则

右键菜单中选择「💾 导出规则」可将所有规则保存为 JSON 文件；选择「📥 导入规则」可从文件恢复规则配置。

4支持的字段类型

类型 说明 数据格式
图片 自动上传图片到表单 图片 URL 或 Base64 字符串
文本 填充普通文本输入框 任意文本字符串
下拉框 选择下拉选项 选项的值或显示文本
文本域 填充多行文本框 任意文本字符串

常见问题

❓ 悬浮按钮不显示？

1. 检查油猴扩展是否已启用
2. 确认当前网页地址与脚本匹配
3. 按 F12 打开控制台查看是否有错误信息

❓ 图片上传失败？

1. 确认图片 URL 可正常访问
2. 检查图片是否支持跨域访问

❓ 如何找到正确的目标字段名称？

在表单页面右键点击悬浮按钮 → 「预览匹配」，脚本会高亮显示所有可识别的字段，根据提示填写正确的目标名称。

⚠️ 注意事项
1. 首次使用前务必先配置映射规则
2. 建议在测试环境先熟悉功能再正式使用
3. 填充前请确认 JSON 数据格式正确
